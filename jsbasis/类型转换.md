### 类型转换

### 转成 boolean

- number 除了+-0，NaN都为true
- string 除了空字符串都为true
- undefined null FALSE
- 引用类型 TRUE

### 转成 string

- number 0 => '0'
- Boolean/function/Symbol => "Boolean/function/Symbol"
- 数组 [1,2] => '1,2'
- 对象 [object, Object]

### 转成 number

- string '1' => 1, 's' => NaN
- 数组 [] => 0, 存在一个元素且为数字时转数字，其他情况 NaN
- null 0
- 除了数组的引用类型 NaN
- Symbol 报错
