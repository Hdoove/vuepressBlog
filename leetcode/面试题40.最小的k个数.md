### 面试题40.最小的k个数

[题目地址](https://leetcode-cn.com/problems/zui-xiao-de-kge-shu-lcof/)

```javascript
/**
 * @param {number[]} arr
 * @param {number} k
 * @return {number[]}
 */
var getLeastNumbers = function(arr, k) {
    if(k===0) return [];
    const sortArr = arr.sort((a,b) => a-b);
    return sortArr.splice(0, k);
};
```
